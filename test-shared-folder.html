This page helps you test if the login/signup button is properly hidden when the URL contains 'shared-folder'.

Test Cases

Test 1: Regular URL (should show login button)

URL: http://localhost/dashboard

Expected: Login button should be visible

Test 2: Shared Folder URL (should hide login button)

URL: http://localhost/shared-folder/8d1aabf8fc432442407b8b50a788451c?folder=jaikalima%252FJaidurgamaa

Expected: Login button should be hidden

Test 3: Another Shared Folder URL (should hide login button)

URL: http://localhost/shared-folder/abc123

Expected: Login button should be hidden

Implementation Details

The implementation adds a computed property isSharedFolderUrl that checks if the current URL pathname contains 'shared-folder'.

In the template, the login button is conditionally rendered with v-if="!isSharedFolderUrl".
